# 功能

自动跳跃到从电脑版 QQ 聊天消息内点击的网址，跳过脑残的电脑管家拦截

# 安装

需要浏览器装有 [Tampermonkey](https://tampermonkey.net/) 插件，下方表格中挑一个链接安装

**注意事项**

- ⚠网页若指向文件会自动开始下载，该插件是为了偷安全网址的懒的，若下载到什么奇奇怪怪的东西我可不管
